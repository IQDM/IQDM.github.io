IMRT QA Data Mining (IQDM)

A project of the AAPM IMRT Working Group

Welcome to IQDM

The intent of the IMRT QA Data Mining project is to provide the medical physics community large-scale, quantitative context to IMRT QA data. This is an official project of the AAPM WG-IMRT.

Members Only

This is a private GitHub project to host data mining code, track our progress, and communicate with other members. If you are a project member, create a (free) GitHub account and provide your user handle to @cutright through the WG-IMRT.
